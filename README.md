# AI Data Analyst Agent

## Overview
An AI-powered data analyst that answers business questions from datasets using LLMs.

## Problem
Businesses struggle to quickly extract insights from raw data without technical expertise.

## Solution
This agent allows users to ask natural language questions and receive data-driven insights instantly.

## Features
- Natural language querying
- Automated insights generation
- Dataset summarization
- (Future) Zoom meeting integration

## Tech Stack
- Python
- OpenAI API
- Pandas
- Streamlit (if used)
